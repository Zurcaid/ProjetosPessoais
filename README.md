## BCI

Código do TCC que desenvolvi na UFPB, o título do código "fft_arduino" é bem auto explicativo, na etapa final o trabalho precisava do processamento digital de sinais eletroencefalográficos para acionamento de uma carga, o que foi feito utilizando um arduino.

## GuitarVillain

Um programa simples que fiz por passatempo, é um jogo no estilo de guitar hero em python para rodar no terminal.

O projettem doiso possui dois códigos, o primeiro deles sendo o "GuitarVillain.py", que roda o jogo no terminal. O outro, "RecorderVillain,py" grava as teclas pressionadas durante a duração da música para depois poder ser utilizado no jogo.
